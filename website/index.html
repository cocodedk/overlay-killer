✕ Overlay Killer فارسی

Kill Overlays. Browse Freely.

Overlay Killer removes cookie consent banners, newsletter popups, and scroll-blocking overlays automatically — on Chrome and Firefox. No configuration needed.

Download for Chrome Download for Firefox

Features

  • Automatically removes cookie consent overlays and privacy banners
  • Restores scrollbars blocked by overlay scripts
  • Works on Chrome (Manifest V3) and Firefox
  • No configuration required — installs and works immediately
  • No data collection — fully private and offline

Installation — Chrome

  1. Download overlay-killer-chrome.zip and extract it.
  2. Open Chrome and navigate to chrome://extensions/.
  3. Enable Developer mode (top-right toggle).
  4. Click Load unpacked and select the chrome/ folder.

Installation — Firefox

  1. Download overlay-killer-firefox.zip and extract it.
  2. Open Firefox and navigate to about:debugging.
  3. Click This Firefox → Load Temporary Add-on.
  4. Select firefox/manifest.json from the extracted folder.

Build from Source

git clone https://github.com/cocodedk/overlay-killer.git
cd overlay-killer
# Load chrome/ or firefox/ directory as appropriate

No build step required — the extension runs directly from source.
